# Aero Pendulum Control (EECE 460)

Control-systems project for a 1-DOF aeropendulum, covering modeling, identification, and controller design.

## Contents

- `final_report.pdf`: complete technical report.
- `final_code.m`: MATLAB implementation used in analysis/control workflow.
- `block_diagram.png`: project system architecture overview.

## Focus Areas

- Dynamic system modeling and parameter identification.
- Controller design and validation through simulation/experiments.
